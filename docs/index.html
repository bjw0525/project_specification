블로그 프로젝트 소개

Spring Boot 기반의 블로그 시스템으로 게시글, 댓글, 좋아요, 동영상(Stories) 등 핵심 기능을 제공하는 웹 애플리케이션입니다.

제작 목적

  • Spring 생태계 전반(웹, JPA, MyBatis, 보안)의 실전 통합 학습
  • 컨텐츠 CRUD, 파일 업로드, 권한 제어 등 웹 서비스 필수 기능 구현
  • 동영상 콘텐츠(Stories)로 미디어 처리 흐름과 UI/UX 실습
  • 포트폴리오 목적의 운영형 프로젝트 아키텍처 설계

프로젝트 개요

  • 플랫폼: Spring Boot 3.5.7, Java 17, Maven
  • 서버: 내장 Tomcat 10.1.48, 포트 8080
  • 데이터베이스: MySQL 8.0 (운영), H2 (테스트)
  • 뷰: Thymeleaf, 정적 리소스 분리(JS/CSS/이미지/동영상)
  • 보안: Spring Security(Form Login, Remember-Me)

주요 기능

  • 게시글/스토리 작성·수정·삭제(ADMIN), 공개 조회
  • 댓글, 대댓글(Posts), 좋아요(AJAX), 조회수
  • 카테고리, 태그, 검색(FULLTEXT)
  • 동영상 업로드(≤100MB), 썸네일 자동 생성
  • 공지사항, 방문자 통계, 사이트 설정
