Getting Started

블로그 프로젝트 소개

Spring Boot 기반의 블로그 시스템으로 게시글, 댓글, 좋아요, 동영상(Stories) 등 핵심 기능을 제공하는 웹 애플리케이션입니다.

Why we build

제작 목적

Spring 전 영역을 아우르는 실전형 학습과 포트폴리오 완성을 목표로 합니다.

  • Spring 생태계 전반(웹, JPA, MyBatis, 보안) 통합 학습
  • 컨텐츠 CRUD, 업로드, 권한 제어 등 필수 기능 구현
  • Stories 동영상 파이프라인으로 미디어 처리·UX 실습
  • 운영형 아키텍처 설계 경험과 포트폴리오 확보
Overview

프로젝트 개요

플랫폼·서버·DB·뷰·보안 구성을 한 곳에 정리했습니다.

  • 플랫폼: Spring Boot 3.5.7, Java 17, Maven
  • 서버: 내장 Tomcat 10.1.48, 포트 8080
  • DB: MySQL 8.0 (운영), H2 (테스트)
  • 뷰: Thymeleaf + 정적 리소스 분리(JS/CSS/이미지/동영상)
  • 보안: Spring Security (Form Login, Remember-Me)
Highlights

주요 기능

실제 운영 블로그에 필요한 기능을 모듈 단위로 구현했습니다.

  • 게시글/스토리 작성·수정·삭제(ADMIN) 및 공개 조회
  • 댓글·대댓글, 좋아요(AJAX), 조회수
  • 카테고리, 태그, 검색(FULLTEXT)
  • 동영상 업로드(≤100MB) + 썸네일 자동 생성
  • 공지사항, 방문자 통계, 사이트 설정
