사전 준비
Checklist

사전 준비

필수 런타임과 포트를 먼저 확보하세요.

  • Java 17, Maven 3.x (Wrapper 포함)
  • MySQL 8.0 (기본 포트 3306, 스키마 blog)
Step 1

데이터베이스 생성

UTF8MB4 문자셋으로 스키마를 초기화합니다.

CREATE DATABASE IF NOT EXISTS blog
  DEFAULT CHARACTER SET utf8mb4
  COLLATE utf8mb4_unicode_ci;
Step 2

테이블 및 테스트 데이터

DDL과 샘플 데이터를 순차 실행합니다.

# 테이블 생성
mysql -u root -p blog < create_tables.sql

# (선택) 테스트 데이터 삽입
mysql -u root -p blog < test_data.sql
Step 3

application.properties 확인

JDBC 연결 정보와 자격 증명을 맞춰 둡니다.

spring.datasource.url=jdbc:mysql://localhost:3306/blog?serverTimezone=UTC&characterEncoding=UTF-8
spring.datasource.username=root
spring.datasource.password=1111
Step 4

애플리케이션 실행

OS 별 Maven Wrapper 명령으로 바로 부팅합니다.

# Windows
mvnw.cmd spring-boot:run

# Linux/Mac
./mvnw spring-boot:run
  • 접속: http://localhost:8080
  • 관리자: admin / 1111 (DataInitializer)
Tips

참고

운영 전환 시 필수로 점검할 항목입니다.

  • 비밀번호 인코더를 BCrypt로 전환
  • 파일 업로드 경로: /uploads/images/, /uploads/videos/
  • H2는 테스트 용도 전용
