기술 스택

Back-End

  • Spring Boot 3.5.7, Spring Framework 6.x
  • Spring Web (MVC): REST 컨트롤러 및 서버 사이드 렌더링
  • Spring Validation(JSR-380): DTO/폼 검증
  • Spring Security: 인증/인가, Form Login, Remember-Me

Persistence

  • Spring Data JPA + Hibernate (MySQLDialect, DDL auto update)
  • MyBatis Spring Boot Starter 3.0.5 (mapper XML, camelCase ↔ snake_case)
  • HikariCP: maxPool=10, minIdle=5, idleTimeout=300s, connTimeout=20s, maxLifetime=1200s
  • DB Driver: MySQL Connector/J (runtime), H2 (test)

Front-End

  • Thymeleaf + thymeleaf-extras-springsecurity6
  • 정적 리소스: /css/**, /js/**, /images/**, /uploads/**
  • JS 모듈: posts(detail/edit/write), stories(list/detail/edit/write)
  • Kakao Maps JavaScript API (키: kakao.maps.javascript.key)

Dev & Build

  • Java 17, Maven 3.x (Wrapper 포함)
  • Spring Boot DevTools: 핫 리로드
  • Lombok: 보일러플레이트 제거(Annotation Processor)
  • IDE: VS Code / IntelliJ IDEA

파일 업로드

  • 멀티파트: 파일/요청 크기 최대 100MB
  • 경로: 이미지 /uploads/images/, 동영상 /uploads/videos/
  • 스토리: 동영상 전용, 썸네일 자동 생성(첫 프레임)
# application.properties (발췌)
spring.servlet.multipart.max-file-size=100MB
spring.servlet.multipart.max-request-size=100MB
