API & Security

Back-End

Spring Boot 3 기반의 REST + 서버 사이드 렌더링 듀얼 아키텍처.

  • Spring Boot 3.5.7 & Spring Framework 6.x
  • Spring Web MVC: REST 컨트롤러와 Thymeleaf 렌더링 병행
  • JSR-380 Bean Validation으로 DTO/폼 입력 즉시 검증
  • Spring Security(Form Login, Remember-Me)로 인증/인가 계층화
데이터 액세스 전략

Persistence

ORM + SQL 매퍼를 병행해 CRUD와 튜닝 모두를 커버.

  • Spring Data JPA + Hibernate(MySQLDialect, DDL auto update)
  • MyBatis 3.0.5: XML 매퍼, camelCase ↔ snake_case 자동 매핑
  • HikariCP(10/5 pool, idleTimeout 300s, connTimeout 20s, maxLifetime 1200s)
  • 드라이버: MySQL Connector/J 운영, H2 테스트 격리
뷰 & 사용자 경험

Front-End

템플릿과 정적 리소스를 분리해 캐싱·배포 효율을 확보.

  • Thymeleaf + springsecurity6 extras로 권한 기반 표현 제어
  • 정적 경로: /css/**, /js/**, /images/**, /uploads/**
  • Posts/Stories 기능별 JS 모듈화(detail, edit, write 등)
  • Kakao Maps JavaScript API (kakao.maps.javascript.key) 연계
Developer Experience

Dev & Build

빌드 자동화와 핫 리로드로 반복 작업을 최소화.

  • Java 17 + Maven 3.x Wrapper로 환경 편차 제거
  • Spring Boot DevTools: 저장 즉시 서버 재시작
  • Lombok으로 DTO/엔티티 보일러플레이트 축소
  • IDE: VS Code / IntelliJ IDEA 병행

파일 업로드

100MB 제한 내에서 이미지·동영상을 안전하게 처리.

  • 멀티파트 한도: 파일/요청 각각 100MB
  • 저장 경로: 이미지 /uploads/images/, 동영상 /uploads/videos/
  • Stories: 업로드 시 첫 프레임 썸네일 자동 생성
